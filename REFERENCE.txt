   → state.playing toggled

   → Vinyl starts spinning (CSS animation)
   → Waveform becomes active
   → Audio plays from YouTube

3. Countdown hits 0 (after song ends)
   → fetchRadio() called
   → New song selected from SONGS
   → New youtubeId received from server
   → player.cueVideoById(newId) called
   → Countdown reset to new video duration
   → Auto-play starts next song

┌─────────────────────────────────────────────────────────────────────────────┐
│ YOUTUBE IFRAME API - KEY METHODS YOU'LL USE                                │
└─────────────────────────────────────────────────────────────────────────────┘

player.cueVideoById(videoId)
  → Load a video without playing
  → Use this in fetchRadio()

player.playVideo()
  → Start playing the cued video
  → Use this in togglePlay()

player.pauseVideo()
  → Pause the video
  → Use this in togglePlay()

player.setVolume(0-100)
  → Set volume level
  → Use in volume slider handler

player.getDuration()
  → Get video length in seconds
  → Use to sync countdown

player.getCurrentTime()
  → Get current playback position
  → Optional: use for waveform visualization

player.getPlayerState()
  → Return: -1 (unstarted), 0 (ended), 1 (playing), 2 (paused), 3 (buffering)
  → Use to check if video finished

┌─────────────────────────────────────────────────────────────────────────────┐
│ ERROR HANDLING - WHAT CAN GO WRONG                                         │
└─────────────────────────────────────────────────────────────────────────────┘

Error: "player is not defined"
  → Cause: player.playVideo() called before onYouTubeIframeAPIReady()
  → Fix: Check if (playerReady && player) before calling

Error: "No audio plays"
  → Cause: youtubeId missing or wrong, video blocked, CORS issue
  → Fix: Check console.log(), verify video ID on youtube.com

Error: "Countdown doesn't match video length"
  → Cause: duration field missing from song
  → Fix: Add duration (in seconds) to each SONGS entry

Error: "Volume slider doesn't work"
  → Cause: No event listener added to #volume element
  → Fix: Add volume handler code after line 981

Error: "Waveform stays still"
  → Cause: state.playing not set correctly
  → Fix: Verify togglePlay() is called from button click
